# Free-To-Play

Free-To-Play is a website show informations about big amount of games 

## Tech Stack

Next.js , React , TypeScript , Redux Toolkit , MaterialUI5

## Tech Stack
live demo :
https://freetoplay-three.vercel.app/ 

## Run Locally

Clone the project

```bash
  git clone https://github.com/mohammadazhari/FreeToPlay
```

Install dependencies

```bash
  npm install
```

Start the app

```bash
  npm run dev
```
